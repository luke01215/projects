## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Usage

1. **Flashcards**: Click on cards to flip and reveal answers. Rate your confidence (Hard/Medium/Easy) to optimize review scheduling
2. **Quiz**: Select a domain or take a comprehensive quiz. Get instant feedback with explanations
3. **Progress**: Track your study statistics and identify areas that need more focus
4. **Reference**: Browse all exam topics organized by domain

## Study Tips

- Study consistently for 30-60 minutes daily
- Focus on domains where your accuracy is below 70%
- Review flashcards marked as "Hard" more frequently
- Take practice quizzes regularly to test understanding
- Use the Reference guide to deepen knowledge on specific topics

## Technology Stack

- React 18 with TypeScript
- Vite for fast development and building
- Tailwind CSS for styling
- Local Storage for progress persistence
